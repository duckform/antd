# duckform antd playground!

## 如何开发
入口文件在 src/playground

playground of [duckform](https://github.com/duckform/duckform)

### get bun
```bash
curl -fsSL https://bun.sh/install | bash
```

```bash
# 安装依赖
bun i
# 构建packages
bun run build
# run dev
bun run dev
```
